# Portfólio Pessoal - Charles Albuquerque

Este é o meu portfólio profissional, desenvolvido para apresentar minhas habilidades como Web Designer e Desenvolvedor. O projeto foi concebido desde a etapa de design visual até a implementação técnica, focando em performance, semântica e acessibilidade.

---

## Processo de Design
Antes de qualquer linha de código, o projeto foi planejado e prototipado no **Figma**. 
> O objetivo do design foi criar uma estética "Premium Dark", utilizando tons de dourado/bege sobre um fundo preto profundo para transmitir elegância e profissionalismo.

---

## Tecnologias Utilizadas

Para este projeto, aceitei o desafio de construir uma interface dinâmica **sem utilizar uma única linha de JavaScript**.

* **HTML5:** Uso rigoroso de tags semânticas para SEO e acessibilidade.
* **CSS3 (Puro):** **Flexbox & CSS Grid:** Para o layout assimétrico de serviços.
    * **CSS Variables:** Para fácil manutenção de cores e fontes.
    * **Media Queries:** Responsividade completa, incluindo suporte para resoluções **4K**, tablets e dispositivos móveis.
    * **Backdrop Filter:** Efeito de desfoque (blur) na navegação estilo "glassmorphism".

---

## Funcionalidades e Destaques

* **Header Hero:** Imagem de fundo com posicionamento inteligente e texto sobreposto para impacto visual imediato.
* **Grid de Serviços Assimétrico:** Layout moderno onde os blocos alternam tamanhos, fugindo do grid padrão.
* **Skills Estáticas:** Barras de progresso elegantes com controle de largura via variáveis CSS.
* **Formulário Funcional (Pure CSS):** Uso do `type="reset"` para limpar os campos após a solicitação, simulando uma interação de envio sem scripts.
* **Responsividade Ultra-Wide:** Otimização específica para monitores 4K para evitar que o layout pareça pequeno em telas grandes.
